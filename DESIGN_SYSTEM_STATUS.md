## Support & Questions

### Troubleshooting
- Check `DESIGN_SYSTEM_MODERN_REFACTOR.md` for concepts
- Check `IMPLEMENTATION_GUIDE.md` for patterns
- Review `components-modern.css` for class names
- Check `/components` page (live showcase)

### Common Issues
1. **Components not styled?** — Ensure you're importing globals.css
2. **Dark mode not working?** — Use theme toggle in header
3. **Animations janky?** — Check if `prefers-reduced-motion` is set
4. **Focus rings missing?** — All elements should have focus-visible
5. **Responsive not working?** — Check breakpoint media queries
