## ⚠️ Key Rules (Don't Break These)

1. **Direction C always** — If it doesn't fit Illustrated Glass (teal/orange/organic), don't use it
2. **TLS components first** — Don't redesign; import Button, Card, etc. from library
3. **One screen at a time** — HOME → FORMATION → ACCOMPAGNEMENT → etc. in order
4. **Annotate as you go** — Notes on Moodboard + checklist updates = your design diary
