## 💡 Pro Tips for Searching

✅ **DO:**
- Save good finds to mood board (screenshot + link)
- Note the specific animation used
- Check hover/interaction states
- Look at multiple pages (scroll entire site)
- Note color combinations
- Screenshot component details

❌ **DON'T:**
- Copy exact designs (inspire, not replicate)
- Use AI-generated images (Direction C is authentic)
- Focus only on static screens (animation is key)
- Ignore mobile view (test responsiveness)
- Overlook micro-interactions (details matter)

## 📋 Checklist: What to Look For

When browsing references:

- [ ] How do elements enter on scroll? (fade? slide? morph?)
- [ ] What's the parallax speed? (fast? slow? layered?)
- [ ] What colors are used? (warm? cool? mixed?)
- [ ] How smooth is the animation? (snappy? floaty? spring-like?)
- [ ] What illustration style? (watercolor? painted? vector?)
- [ ] Are there floating elements? (particles? shapes? blobs?)
- [ ] What's the glass effect like? (subtle? strong? layered?)
- [ ] How does hover work? (lift? glow? scale? rotate?)
- [ ] What loading states are shown? (shimmer? skeleton? spinner?)
- [ ] How's the typography animated? (slide? reveal? morph?)
