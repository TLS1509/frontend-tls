## PROPOSALS COPY — Hero (2026-06-10)

> **Contexte** : analyse concurrentielle juin 2026 montre que « L'IA augmente » est générique
> (360Learning, Didask, Degreed, France Compétences disent tous une variante). Notre edge
> n'est pas la *posture IA* — c'est **QUI on augmente** (le formateur) et **la boucle de preuve**
> (Learn→Do→Match + passeport). Ces proposals restent à valider par Chloé.
>
> **3 problèmes diagnostiqués dans la copy actuelle :**
> - P1 — H1 générique : « L'IA augmente vos compétences. Pas l'inverse. » → tout le monde dit ça.
> - P2 — Em dash interdit (MARQUE-VOIX) dans le sous-titre + « humain au centre » = cliché universel.
> - P3 — Cadence aphoristique (énoncé + courte négation) apparaît 3× sur la page (H1 / Conviction / §7). MARQUE-VOIX : au-delà de 2, réécrire.

### H1 — 5 options

**H1-A — Catégorie vacante (recommandée)**
```
Le programme qui augmente le formateur.
Pas juste ses apprenants.
```
Rationale : nomme explicitement le terrain vacant. IAvenir/SavoirIA forment *des apprenants* ;
TLS forme *le formateur lui-même*. Ligne 2 en primary-600. Fort visuellement.
⚠️ "Pas juste" = encore le pattern aphoristique, mais une fois c'est OK.

---

**H1-B — Concret + double promesse**
```
Maîtrisez l'IA dans votre pratique pédagogique. Prouvez-le avec un Open Badge.
```
Rationale : deux actions concrètes (maîtriser + certifier), FACTS-CANON F6 respecté.
Bon si l'audience arrive sans contexte TLS. Légèrement long pour un H1.

---

**H1-C — Angle preuve vs promesse (éditorial, court)**
```
Des compétences IA prouvées, pas promises.
```
Rationale : adresse la méfiance des formateurs (« formations IA décevantes » verbatim Audience A).
Cohérent avec §7 « La preuve, pas la promesse ». Ultra-court, fort en typo.
⚠️ Ne nomme pas le formateur — compenser dans le sous-titre.

---

**H1-D — Tagline-marque, ultra-court**
```
Le formateur augmenté.
```
Rationale : possède la catégorie en 3 mots. Fonctionne comme titre de marque + SEO fort
sur le terme qu'on veut posséder. Nécessite sous-titre dense. Très fort visuellement.

---

**H1-E — Pair-à-pair, pain point réel (arc MARQUE-VOIX §6)**
```
Vous avez déjà passé 4 heures sur un plan de cours que vos stagiaires n'ont pas lu ?
Voici ce que ça donne avec un assistant IA bien paramétré.
```
Rationale : verbatim tiré de MARQUE-VOIX arc 1, adapté en vous. Ultra-concret, adresse
le pain point temps. Trop long pour un H1 classique — plus fort en landing formation ciblée.

### Conviction stripe (bande teal) — 2 options

**Conviction A — Prend position, nomme le formateur**
```
Nous n'avons pas construit une formation IA pour tout le monde. Nous avons construit
le programme que les formateurs attendaient : apprendre l'IA sur leurs propres projets,
le prouver avec un badge, le valoriser dans leur pratique.
```
Sort du générique. Trois temps (Learn→Do→Match implicites). Pas de cliché.

**Conviction B — Anti-hype littéral (verbatim MARQUE-VOIX §5)**
```
Ce qu'on ne fait pas : des chatbots qui remplacent le formateur, des badges vides,

un projet réel, un Open Badge qui prouve la compétence acquise.
```
Concret, différencie du cluster Qualiopi généraliste. Adresse directement les objections.
⚠️ Double structure « Ce qu'on ne fait pas / Ce qu'on fait » peut lasser si répété ailleurs.
