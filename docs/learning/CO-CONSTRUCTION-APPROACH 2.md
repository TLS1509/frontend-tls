# Co-Construction Approach — Learning App + SBO + Site (12 Weeks)

> **Philosophy**: This isn't a course you watch passively. This is **hands-on, collaborative building**. Every lesson, every week, every project is built **together** with me as your coding partner, design critic, and technical guide.

---

## 🤝 How We Work Together

### Week-by-Week Collaboration Model

**Each week follows this rhythm:**

1. **Monday Planning (30 min)**
   - Review the week's Étape in the Learning App
   - Clarify objectives (what will we have by Friday?)
   - Discuss blockers from last week
   - Git branch: `bootcamp/week-N`

2. **Daily Build (Tue–Thu, 4–6h focused work)**
   - You code in your editor, I code alongside you
   - Real-time problem-solving: stuck on layout? We debug together
   - Design review: screenshot → feedback → iterate
   - Git commits together (co-authored)

3. **Friday Review (2h)**
   - Ship what we built (merge to main)
   - QA: test the week's deliverables
   - Document learnings (update SESSION notes)
   - Celebrate shipping ✓

4. **Weekend Reflection (1h)**
   - Retrospective: what worked, what to improve
   - Prepare next week's plan
