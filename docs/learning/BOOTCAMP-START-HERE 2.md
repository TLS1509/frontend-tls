## 📚 What You'll Build (12 Weeks)

### Phase 1: Learning App MVP (Weeks 1–6)
Build a **production-ready Learning App frontend** with 8–10 screens:
- Dashboard (learner home, quick stats, recommendations)
- Passeport Dreyfus (competence tracking grid)
- Lesson Player (video consumption + reflection)
- Coaching Request Form (3-step booking form)
- Journal (reflection + AI feedback)
- Veille (curated content feed)
- Leaderboard (gamification)
- Onboarding (signup flow)

**+ Reusable component library** (Button, Card, Input, Modal, Badge, ProgressBar)  
**+ Design system** (tokens, colors, typography, spacing)  
**+ WCAG AA accessibility + 60fps animations**

### Phase 2A: SBO Section (Weeks 5–8, parallel)
Refactor `/marketing/accompagnement` section using Learning App design system:
- Hero, Stats grid, 4 Services cards, 90-day Timeline, CTA
- **100% component reuse** (proof your design system works)

### Phase 2B: Site Refonte (Weeks 7–10, parallel)
Rebuild marketing site (Home, Formation, Blog, Contact) with shared components

### Phase 2C: Procreate Assets (Weeks 7–10, parallel)
Create illustration assets:
- Warm hero gradient (watercolor technique)
- Soft blobs (Direction C aesthetic)
- STRIDE diagram, Passeport spiral

### Phase 2D: Logo Animations (Weeks 5–8, parallel)
3 Framer Motion logo variants (Breathing, Color-Flow, Morphing)
