### Why Parallel?
- **Procreate** = right-brain, creative (iPad, hands-on)
- **Framer Motion** = left-brain, analytical (desktop, coding)
- Switching between them keeps you fresh
- Both skills are independent (can learn in any order)

## 🚨 Common Questions

**Q: Can I do Procreate → Framer Motion sequentially (not parallel)?**  
A: Yes! Pick Path 2 above. Just know Procreate requires iPad hands-on time, so days 1–5 are iPad-heavy, days 6–7 are code-heavy.

**Q: What if I already know Framer Motion?**  
A: Skip `FRAMER-MOTION-LEARNING.md`. Just copy the `AnimatedLogo.tsx` code from `ANIMATED-LOGO-PROJECT.md` and focus on `PROCREATE-BOOTCAMP.md`.

**Q: What if I get stuck on a task?**  
A: Each doc has a "Troubleshooting" section at the bottom. If still stuck, refer to external resources (YouTube links, official docs, community forums).

**Q: When should I start Week 2?**  
A: After completing Week 1 (all deliverables done + comfort ≥ 8/10 on both tools). Week 2 covers STRIDE diagram, Passeport spiral, SBO infographic (Procreate), and video production (Framer Motion or vendor).

**Q: Can I use the code snippets as-is?**  
A: Yes! All code is copy/paste ready. But reading the explanations first will help you understand *why* the code works (not just *how*).
