### Hands-on Practice Tasks

**Task 1.1 : Canvas Setup (15min)**
```
1. Open Procreate
2. Create NEW canvas:
   - Size: 1920 × 1080 pixels
   - Color space: sRGB
   - DPI: 300 (for print quality exports)
   - Name: "Hero-Mesh-Gradient"
3. Save to iCloud Drive → Projects folder
4. Explore zoom (two-finger pinch, 3-finger tap)
```

**Task 1.2 : Layer Exploration (30min)**
```
1. Create a new layer (top-left icon, or swipe right with two fingers)
2. Rename it "Sketches"
3. Create 3 more layers: "Base", "Mid-tones", "Highlights"
4. Group them: select all 3 → tap layer header → "Group"
5. Name group "Watercolor Pass 1"
6. Toggle visibility (eye icon) on/off for each layer
7. Test blend modes (tap "N" on a layer):
   - Try: Multiply, Screen, Overlay (see differences)
   - Notice how each blends with layers below
8. Delete one layer to practice undo/redo (three-finger tap)
```

**Task 1.3 : Brush Pressure Curve (20min)**
```
1. Go to Brush Settings (wrench icon)
2. Open a brush (e.g., "Sketching Pencil")
3. Tap "Brush Dynamics" or "Pressure"
4. Look at the pressure curve graph (diagonal line)
   - Drag it to change responsiveness
   - Steep = sensitive to light pressure
   - Flat = requires strong pressure
5. Draw 20 pressure-sensitive strokes on your canvas:
   - Light touch (curved line)
   - Medium pressure (medium weight)
   - Heavy pressure (thick line)
6. Observe how the brush weight changes

💡 TIP: Try "Uniform Pressure Curve" first, then experiment with custom curves later.
```

**Task 1.4 : Color Picker (15min)**
```
1. Tap + hold on a color swatch (bottom-left) to color picker
2. Sample colors from reference images:
   - Use eyedropper (left icon in color picker)
   - Tap on reference image on another screen
   - Save to swatches (tap + hold swatch area)
3. Create a palette with:
   - Teal #55A1B4
   - Pêche #F5B8A8
   - Gold #F8B044
   - Cream #FDF8F3
4. Save this palette (long press palette area)
   - Name: "Direction-C-Palette"
```

**Task 1.5 : First Strokes (20min)**
```
1. Select a brush (try "Charcoal" or "Pencil")
2. Draw 10 simple shapes:
   - Circle
   - Square
   - Triangle
   - Wavy line
   - Spiral
3. Use different pressures for each
4. On a new layer, draw the same shapes with a different brush
5. Compare: how does the brush personality change the mark?
```

### Procreate Watercolor Brushes to Explore
```
In Brush Library, find:
✓ "Watercolor" (category) — default set
  └─ "Wet Watercolor" (best for diffusion)
  └─ "Splatter" (texture/organic)
✓ "Liquid" (category)
  └─ "Ink Wash" (wet bleeds, perfect for blobs)
✓ "Paint" (category)
  └─ "Round Brush" (smooth strokes)
  └─ "Oil Brush" (thicker coverage)

💡 TIP: Favorite your top 5 brushes by tapping the heart icon.
```

### Hands-on Practice Tasks

**Task 3.1 : Wet-on-Wet Technique (45min)**
```
1. Create a new canvas: 1200 × 800 (landscape)
2. Create layer "Wet Pass"
3. Select brush: "Ink Wash" from Liquid category
4. Paint a solid block of color (e.g., Teal) covering ~60% of canvas
5. WITHOUT CLEARING the layer, switch to "Wet Sponge" brush
6. Dip into the same color (or adjacent color)
7. Brush over the wet area — notice it DIFFUSES (soft edges)
8. Try another color (Pêche) overlapping the teal
   - Notice: colors blend where they meet (additive mixing)
9. Reduce opacity of "Wet Pass" layer to ~50%
10. Add new layer below: "Base"
11. Paint background color (Cream) on Base layer
12. Observe how 50% opacity Wet Pass creates transparency effect
13. Save as "Watercolor-Test-1.procreate"
```

**Task 3.2 : Color Blending with Blend Modes (30min)**
```
1. Create new canvas: 1000 × 1000
2. Create layer "Base Color" — fill with Cream
3. Create layer "Multiply Pass":
   - Change blend mode to "Multiply"
   - Paint dark Teal in the center
   - Result: darkens Base, feels like shadow
4. Create layer "Screen Pass":
   - Change blend mode to "Screen"
   - Paint light Gold at top
   - Result: lightens Base, feels like light
5. Create layer "Overlay Pass":
   - Change blend mode to "Overlay"
   - Paint Pêche on the side
   - Result: both darkens AND lightens (complex blend)
6. Experiment: toggle each layer on/off to see the effect
7. Key insight: blend modes are "mixing filters" for digital watercolor
```

**Task 3.3 : Layered Gradient (45min)**
```
1. Create new canvas: 1400 × 1400 (square, for blob work)
2. Create 3 new layers: "Layer-Teal", "Layer-Peche", "Layer-Gold"
3. On Layer-Teal:
   - Select "Wet Watercolor" brush
   - Set opacity to 40%
   - Paint 3–4 organic blobs in bottom-left area
   - Colors: Teal + darker Teal (shadow side)
4. On Layer-Peche:
   - Opacity: 35%
   - Paint 2–3 blobs overlapping Teal area (center)
   - Colors: Pêche + orange (warm transition)
5. On Layer-Gold:
   - Opacity: 45%
   - Paint 2–3 blobs overlapping Pêche (top-right)
   - Colors: Gold + cream (light transition)
6. Step back: you've created a "gradient" using layered blobs
   - This is the core technique for Hero mesh gradient!
7. Save as "Gradient-Test.procreate"
```

**Task 3.4 : Texture & Splatter (20min)**
```
1. Create new canvas: 800 × 600
2. Paint a base color wash (Teal, Wet Watercolor, 50% opacity)
3. Create new layer "Texture"
4. Select "Splatter" brush (or "Dust & Scratches")
5. Set size to ~50px
6. Tap 5–10 times around the canvas (creates splatter texture)
7. Reduce layer opacity to 20% (so it's subtle, not overwhelming)
8. Create another layer "Grain"
9. Select "Watercolor Texture" or "Paper Grain" brush
10. Paint across entire layer (creates fine grain texture)
11. Reduce opacity to 15%
12. Result: base color + splatter + grain = organic watercolor feel
```

### Checkpoint ✅
- Can paint wet-on-wet and see diffusion ✅
- Understand blend modes (Multiply = shadow, Screen = light) ✅
- Created a 3-color gradient using layered blobs ✅
- Splatter + grain feels natural, not overdone ✅

### Hands-on: Hero Mesh Gradient Asset

**Task 5.1 : Create Hero Mesh Gradient (4–6 hours)**

```
SPECS:
├─ Canvas: 1920 × 1080px @ 300dpi
├─ Filename: "Hero-Mesh-Gradient.procreate"
├─ Palette: Teal #55A1B4 · Pêche #F5B8A8 · Gold #F8B044 · Cream #FDF8F3
├─ Reference: Phantom (organic blobs), Structured (baroque feel)
└─ Opacity target: 40–60% (sits behind glass components)

WORKFLOW:
```

**Step 1: Sketch (30min)**
```
1. Create layer "Sketch"
2. Select light pencil brush
3. Lightly sketch 5–7 blob shapes:
   - One large teal blob (bottom-left)
   - One pêche blob (center, overlapping)
   - One gold blob (top-right)
   - Fill remaining space with smaller blobs
   - Reference Phantom for organic, non-geometric shapes
4. Don't commit yet — just rough guides
```

**Step 2: Base Color Pass (45min)**
```
1. Create layer "Base Color" (below Sketch)
2. Select "Wet Watercolor" brush
3. Set opacity to 50%
4. Paint Teal in bottom-left area:
   - Start with pure Teal
   - Blend toward darker Teal (shadow side)
   - Use Wet Sponge to diffuse edges
5. Switch to Pêche:
   - Paint center area (overlapping Teal slightly)
   - Blend with orange tones
6. Switch to Gold:
   - Paint top-right area
   - Blend with cream (highlight side)
7. Fill gaps with Cream
8. Result: rough 3-color gradient
```

**Step 3: Refine Blending (1 hour)**
```
1. Create layer "Blend Pass" (above Base)
2. Set blend mode to "Overlay"
3. Set opacity to 30%
4. Paint with subtle color transitions:
   - Where Teal → Pêche: add coral/salmon tones
   - Where Pêche → Gold: add warm orange
   - Soften hard edges
5. Create layer "Shadows" (above Blend):
   - Blend mode: Multiply
   - Opacity: 20%
   - Paint darker version of each color at blob edges
   - Creates depth (shadow side vs light side)
6. Create layer "Highlights" (top):
   - Blend mode: Screen
   - Opacity: 25%
   - Paint light cream/white at top-left
   - Creates light direction illusion
```

**Step 4: Texture & Detail (45min)**
```
1. Create layer "Texture" (above everything)
2. Select "Watercolor Texture" or "Splatter" brush
3. Paint splatter across entire canvas:
   - 30–50% opacity (subtle, not dominating)
   - Varies splatter size (large + small spots)
4. Create layer "Paper Grain":
   - Select brush: "Paper Grain" or "Watercolor Texture"
   - Paint light strokes across entire canvas
   - 10–15% opacity (very subtle)
5. Step back: observe texture + gradient balance
6. If too much texture: reduce opacity
   If not enough: add another pass
```

**Step 5: Final Polish (30min)**
```
1. Flatten image (Adjustments → Flatten Image)
   OR keep layers for Figma import
2. Reduce overall opacity to 50%:
   - Test on white background
   - This is how it'll look behind glass components
3. Adjust color balance if needed:
   - Adjustments → Hue, Saturation, Brightness
   - Teal should feel cool but not cold
   - Pêche should be warm, not orange
4. Save as "Hero-Mesh-Gradient.procreate" (native format, keeps layers)
```

**Task 5.2 : Export Formats (30min)**

```
1. File → Share (or export)
2. PNG (2x retina for web):
   ├─ Name: "hero-mesh-gradient@2x.png"
   ├─ Scale: 2x (outputs 3840 × 2160)

3. PNG (1x for reference):
   ├─ Name: "hero-mesh-gradient.png"

4. PSD (for Figma):
   ├─ Name: "hero-mesh-gradient.psd"
   ├─ Includes: all layers (for color adjustments later)
   └─ Color space: sRGB
5. Move files to: ~/Documents/Claude/Projects/frontend-tls/public/assets/illustrations/

## 💡 Key Lessons Learned

1. **Pressure sensitivity is everything** — Procreate's strength is tactile feedback on iPad
2. **Blend modes are your friends** — Multiply (shadow), Screen (light), Overlay (complex mixing)
3. **Opacity control mimics paint dilution** — 50% opacity = watercolor wash
4. **Layering > single stroke** — Multiple 30% opacity layers read better than one 100% layer
5. **Splatter + grain are subtle** — 10–20% opacity, not 50%+

## 🚨 Troubleshooting

| Problem | Solution |
|---------|----------|
| Brush strokes too hard/scratchy | Adjust pressure curve (more sensitivity) OR try "Wet Sponge" brush |
| Colors look muddy when blending | Reduce opacity to 30–40%, use Multiply/Overlay blend modes |
| Splatter feels artificial | Reduce opacity to 10–15%, vary splatter size |
| Gradient looks too uniform | Add more blob shapes, use different opacities per layer |
| Canvas too slow/laggy | Reduce canvas resolution (start at 1200×800 for practice) |
| Undo not working | Three-finger tap (undo), two-finger tap (redo) — it's gesture-based |
