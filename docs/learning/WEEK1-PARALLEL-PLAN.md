## 🚨 If You Get Stuck

### Procreate Issues
- **Brush feels too scratchy** → Adjust pressure curve (higher sensitivity)
- **Colors muddy when blending** → Reduce opacity to 30–40%, use Multiply/Overlay
- **Can't find watercolor brushes** → Go to Brush Library → Search "water" or "ink"
- **Canvas too slow** → Reduce resolution (start at 1200×800 for practice)

**Resource** : Open `PROCREATE-BOOTCAMP.md` → "Troubleshooting" section

### Framer Motion Issues
- **Animation feels jerky** → Check DevTools → Performance tab (should be 60fps)
- **AnimatePresence not working** → Verify component is wrapped inside `<AnimatePresence>`
- **SVG not animating** → Check that `<motion.path>` has `pathLength` prop (not SVG wrapper)
- **Variants not working** → Make sure `initial` and `animate` props reference variant names

**Resource** : Open `FRAMER-MOTION-LEARNING.md` → "Troubleshooting" section

## 📚 Quick Reference: Key Commands

### Procreate Shortcuts
```
3-finger tap   = Undo
2-finger tap   = Redo
2-finger drag  = Pan
2-finger pinch = Zoom
Touch + pencil = Color picker (hold)
```


### DevTools Performance Check
```
1. Open browser DevTools (F12)
2. Go to Performance tab
3. Click record (red dot)
4. Interact with animation (hover, click, etc.)
5. Stop recording
6. Look for green line (60fps) throughout timeline
   - Green = smooth
   - Yellow/red = lag (needs optimization)
```
