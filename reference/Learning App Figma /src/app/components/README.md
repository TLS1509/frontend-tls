# Components

Tous les composants réutilisables du projet.

## Structure

```
/components/
├── /ui/              - Primitives base (Button, Card, Dialog...)
├── /common/          - Layout partagé (PageContainer, SectionHeader...)
├── /celebrations/    - Système célébrations
├── /coaching/        - Réservation coaching
├── /modals/          - Modals spécifiques
├── /feedback/        - Collecte feedback
├── /assessment/      - Positionnement compétences
├── /notifications/   - Système notifications
├── /veille/          - Composants veille
├── /journal/         - Journal apprentissage
├── /onboarding/      - Flow onboarding
├── /design-system/   - Composants design system page
├── /patterns/        - Patterns composition
├── /quiz/            - Système quiz
├── /rating/          - Système notation
├── /typography/      - Composants typo
├── /debug/           - Outils debug
└── /figma/           - Imports Figma (protected)
```

## Documentation

Chaque dossier a un `README.md` avec :
- Liste composants
- Exemples import
- Lien doc détaillée

**Documentation complète:** `/docs/02-COMPONENTS.md`

## Règles

✅ Variables CSS uniquement  
✅ Font faces définies uniquement  
✅ 1 README.md par dossier max  
✅ README < 100 lignes
