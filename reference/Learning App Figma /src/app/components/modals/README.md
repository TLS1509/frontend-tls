# Modals Components

Modals spécifiques métier.

## Composants
- `AchievementUnlockModal` - Déblocage badges
- `StreakCelebrationModal` - Célébration séries
- `SuccessModal` - Success générique
- `EmojiRatingModal` - Rating emoji
- `PositionnementModal` - Positionnement compétences
- `BookingConfirmationModal` - Confirmation booking
- `ConfirmationModalAdvanced` - Confirmation avancée

## Import
```tsx
import { AchievementUnlockModal } from './components/modals/AchievementUnlockModal';
```
