# Common Components

Layout et composants partagés.

## Composants

### Layout
- `PageContainer` - Wrapper pages
- `SectionContainer` - Wrapper sections  
- `SectionHeader` - En-têtes sections
- `PageHeaderFinal` - Header pages
- `PageHeaderSimple` - Header simple

### Navigation
- `SearchBar` - Recherche simple
- `SearchBarWithFilters` - Recherche + filtres
- `FilterBar` - Barre filtres
- `AdvancedFilterBar` - Filtres avancés

### UI
- `Button` - Boutons custom
- `Badge` - Badges custom
- `Card` - Cartes custom
- `GlassCard` - Carte glassmorphism
- `InfoAlert` - Alertes info

## Import
```tsx
import { PageContainer } from './components/common/PageContainer';
import { SectionHeader } from './components/common/SectionHeader';
```
