# Animations Components

Composants d'animation et effets visuels.

## Composants

### Emojis Animés
- `EmojiWaveGlass.tsx` - Emoji avec effet glassmorphism
- `EmojiWaveInline.tsx` - Emoji inline avec animation
- `Emoji3D.tsx` - Emoji avec effet 3D

### Icons Animés
- `AnimatedHandIcon.tsx` - Icône main animée

## Import
```tsx
import EmojiWaveGlass from './components/animations/EmojiWaveGlass';
import { AnimatedHandIcon } from './components/animations/AnimatedHandIcon';
```

## Utilisation
Utilisés dans:
- Dashboard Hero
- Pages démo
- Célébrations
