# Typography Components

Composants typographie.

## Composants
- `Heading` - Headings typographiques
- `Text` - Texte typographique

## Import
```tsx
import { Heading, Text } from './components/typography';
```

**Utiliser variables CSS uniquement:**
- `var(--font-display)` pour headings
- `var(--font-body)` pour texte
