## 🚀 Utilisation

### Option 1 : Fichiers HTML Standalone
1. Téléchargez le fichier `.html` souhaité
2. Ouvrez-le directement dans votre navigateur
3. Tout fonctionne immédiatement (styles, fonts, animations)

### Option 2 : Intégration dans un Projet
1. Copiez le contenu HTML du `<body>`
2. Intégrez-le dans votre template
3. Gardez les styles `<style>` ou extrayez-les dans un fichier CSS externe

### Option 3 : Figma Design
1. Utilisez le fichier `figma-design-tokens.json` pour importer les tokens dans Figma
2. Les couleurs, typographies et espacements correspondent exactement

- **Polices** : Google Fonts avec `display=swap`
- **Icons** : Lucide via CDN (léger)
- **CSS** : Minifié et optimisé
- **Animations** : GPU-accelerated (transform, opacity)

## 🎯 Pour Figma Design

### Tokens de Design Exportés
Le fichier `figma-design-tokens.json` contient :
- Palette de couleurs complète (primary, secondary, accent, neutral)
- Échelle typographique (tailles, poids, line-heights)
- Espacements (scale 4pt)
- Radius (6px à 24px)
- Shadows (6 niveaux)

### Import dans Figma
1. Installez le plugin **"Tokens Studio for Figma"**
2. Importez `figma-design-tokens.json`
3. Tous les tokens sont synchronisés avec le code
