## 🚀 Workflows Recommandés

### Workflow 1 : Test Rapide (5 minutes)
```
1. Ouvrez LISEZ-MOI.txt
2. Double-cliquez sur agence-demo.html
3. Testez dans votre navigateur
✅ Vous avez vu le résultat !
```

### Workflow 2 : Personnalisation (30 minutes)
```
1. Lisez INDEX.md (comprendre le design system)
2. Ouvrez agence-demo.html dans un éditeur
3. Modifiez les textes et couleurs
4. Testez dans le navigateur
5. Créez d'autres pages par duplication
✅ Vous avez personnalisé le site !
```

### Workflow 3 : Design Figma (1 heure)
```
1. Lisez FIGMA-GUIDE.md
2. Installez Tokens Studio plugin
3. Importez figma-design-tokens.json
4. Créez vos composants Figma
5. Exportez et comparez avec agence-demo.html
✅ Design et code synchronisés !
```

### Workflow 4 : Développement Complet (2-3 heures)
```
1. Lisez README-HTML.md (structure technique)
2. Analysez le code de agence-demo.html
3. Extrayez les styles CSS dans un fichier externe
4. Créez les 5 autres pages (home, academie, etc.)
5. Testez le responsive sur tous les écrans
6. Déployez sur votre serveur
✅ Site complet déployé !
```
