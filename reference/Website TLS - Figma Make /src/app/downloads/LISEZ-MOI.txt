───────────────────────────────────────────────────────────────────────
🔧 PERSONNALISATION FACILE
───────────────────────────────────────────────────────────────────────

CHANGER LES COULEURS
  1. Ouvrez agence-demo.html avec un éditeur
  2. Cherchez "<style>" au début du fichier
  3. Trouvez "--primary-500: #55A1B4;"
  4. Remplacez par votre couleur

CHANGER LES TEXTES
  1. Cherchez le texte dans le HTML
  2. Remplacez par votre contenu
  3. Sauvegardez

AJOUTER DES ICÔNES
  → Liste complète sur : https://lucide.dev/icons
  → Copiez : <i data-lucide="nom-de-icone"></i>
  → Exemples : sparkles, arrow-right, check-circle

📋 STRUCTURE DU FICHIER HTML
───────────────────────────────────────────────────────────────────────

agence-demo.html contient :

  1. HERO SECTION
     • Badge + Titre + Description
     • Bouton CTA
     • Banner "Philosophie" (glassmorphism)

  2. SERVICES SECTION
     • 4 services en grille
     • 2 grandes cartes + 2 standard
     • Responsive automatique

  3. TIMELINE SECTION
     • 3 étapes (Cadrage, Prototype, Scale)
     • Layout horizontal compact

  4. CTA SECTION
     • Titre + Description
     • 2 boutons CTA
     • Trust indicators
