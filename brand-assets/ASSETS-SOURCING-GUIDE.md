# Guide de sourcing d'assets — The Learning Society

Où trouver photos, vidéos, illustrations et graphiques **alignés à l'identité TLS** (teal/orange/cream, chaleureux, humain). Sélection volontairement **resserrée** : les meilleures sources par catégorie, pas un annuaire. Tout est gratuit/usage commercial sauf mention.

---

## 🎯 North Star (le filtre absolu)

> Des **humains réels en train d'apprendre, collaborer, progresser** — lumière naturelle, vrais lieux de travail, diversité, énergie posée et optimiste. La compétence en action.

### OUI — Ce qu'on cherche

- Binômes / petits groupes qui échangent
- Moment "déclic" — expression de compréhension
- Mains qui écrivent, manipulent, prennent des notes
- Mentor + apprenant — regard bienveillant
- Formateur devant un groupe, engagement réel
- Lumière naturelle chaude (bois, crème, terracotta)
- Espaces de travail modernes et vivants (open spaces, bureaux design, cafés)
- Diversité réelle (pas composée)
- Écrans d'ordinateurs/tablettes utilisés activement (pas mis en scène)
- Couleurs compatibles avec teal (#4A8FA1) et orange (#ED843A)

### NON — Ce qu'on évite

- Poignées de main corporate
- Réunions en costume sans vie
- Personne seule triste devant un laptop
- Clichés "IA cyber" — néons bleus, circuits imprimés, robots métalliques, cerveau 3D
- Stock souriant artificiel (visages trop parfaits, décors blancs aseptisés)
- Filtres bleus/froids (désalignés avec notre palette warm)
- Homogénéité (que des cadres blancs quadragénaires)
- Images à forte dominante rouge/violet (clash avec la palette)

### Palette à viser dans le choix

| Dominante recherchée | Couleurs cibles |
|---|---|
| Fonds / atmosphère | Bois clair, béton chaud, crème, blanc cassé |
| Accents naturels | Feuillage, plantes vertes |
| Éclairage | Lumière naturelle chaude, dorée (pas flash froid) |
| Compatible teal | Bleu-vert ciel, eau, lin gris-vert |
| Compatible orange | Terracotta, cuivre, cuir caramel |

## 📷 Photographie

### Sources primaires (gratuit + usage commercial)

| Source | Pourquoi c'est le bon choix | Termes de recherche TLS |
|---|---|---|
| **[Pexels](https://www.pexels.com)** | Le plus "humain & chaleureux", gratuit usage commercial, sans attribution requise. Grande collection EdTech/formation. | `mentorship`, `team learning`, `workshop collaboration`, `diverse coworkers talking`, `coaching session`, `note taking hands`, `training session`, `professional development` |
| **[Unsplash](https://unsplash.com)** | Qualité éditoriale élevée, gros plans expressifs, collections thématiques curées. | `learning together`, `professional growth`, `candid office warm light`, `study group`, `skill building`, `brainstorming session` |
| **[Reshot](https://www.reshot.com)** | Moins connue = visuels moins vus partout. Style documentaire, authentique. | `real people working`, `team collaboration`, `educator`, `hands laptop` |
| **[Burst by Shopify](https://www.shopify.com/stock-photos)** | Bonnes photos business lifestyle avec lumière chaude. | `entrepreneur`, `team meeting`, `workspace warm`, `learning skills` |

### Recherche efficace — Astuces

- **Filtre par couleur** sur Unsplash/Pexels → choisir "orange" ou "beige" pour des images à dominante warm compatible
- **Orientation** : paysage 16:9 pour hero/slides, carré 1:1 pour social, portrait 4:5 pour mobile stories
- **Résolution** : minimum 2000px de large pour usage web + print
- **Éviter la page 1** : les images les plus populaires sont over-utilisées — aller en page 3+

### Contextes d'utilisation

| Contexte | Style recommandé | Source favorite |
|---|---|---|
| Hero page marketing | Large, atmosphérique, espace pour texte en overlay | Unsplash |
| Card article / vignette blog | Sujet centré, lisible petit | Pexels |
| Témoignage / citation | Portrait proche, regard direct, fond neutre | Reshot |
| Section "équipe TLS" | Style documentaire, naturel, lumière vraie | Séance photo dédiée |
| Social media (Instagram/LinkedIn) | Composition forte, couleurs contrastées | Pexels |
| Fond de présentation slide | Très calme, peu d'éléments, zone sombre pour texte | Unsplash |

## 🎬 Footage / Vidéo

| Source | Pourquoi | Termes |
|---|---|---|

| **[Mixkit](https://mixkit.co/free-stock-video/)** | Clips courts propres (3–15s), parfait b-roll d'intro/transition. Pas d'inscription requise. | `office collaboration`, `students learning`, `hands typing warm`, `presenter speaking` |

**Format recommandé** : MP4 H.264, 1920×1080 minimum (4K si dispo pour future-proofing).

## ✏️ Illustrations

**Règle d'or : une seule source par document/campagne.** Ne pas mélanger les styles (ex. unDraw + Storyset dans la même présentation = incohérent).

### Sources avec personnages (scènes humaines)

| Source | Style | Customisation couleur | Licence |
|---|---|---|---|
| **[Storyset](https://storyset.com)** | Chaleureux, arrondi, beaucoup de scènes EdTech/coaching. **Animations SVG disponibles.** | Éditeur en ligne → coller `#4A8FA1`. | Gratuit avec attribution Freepik. Premium pour retirer. |
| **[unDraw](https://undraw.co)** | Épuré, moderne, flat design. **Open-source, aucune attribution.** | Sélecteur global → `#4A8FA1` → toute la lib passe en teal TLS. | 100% libre, usage commercial illimité. |
| **[Blush](https://blush.design)** | Personnages composables, très diversifiés. | Couleurs éditables directement. | Gratuit (watermark) ou $14/mois. |
| **[Open Peeps](https://www.openpeeps.com)** | Personnages vectoriels modulaires. Idéal pour composer des scènes custom. | Couleur éditable en SVG. | CC0 — domaine public, usage illimité. |
| **[Humaaans](https://www.humaaans.com)** | Personnages modernes, corps + têtes interchangeables. | SVG éditable. | CC0. |

### Sources sans personnages (concepts, data, process)

| Source | Style | Usage TLS |
|---|---|---|
| **[ManyPixels](https://www.manypixels.co/gallery)** | Illustrations conceptuelles flat. Bonne section "Education". | Concepts de formation, parcours, progression |
| **[DrawKit](https://www.drawkit.com)** | Scènes lifestyle et business. Plusieurs packs gratuits. | Présentations commerciales, cas clients |
| **[Sketchvalley](https://sketchvalley.com)** | Style sketch/dessiné à la main, chaleureux. | Supports pédagogiques, journaux, onboarding |

### Utilisation dans l'app React

Les illustrations **SVG** de ces sources peuvent être intégrées directement dans le code.
Storyset et unDraw livrent des SVG recolorisables en Tailwind (`text-primary-500 fill-current`).

## 🌀 Graphiques, fonds, patterns & textures

### Générateurs de fonds (SVG sur mesure)

| Outil | Ce qu'il génère | Comment l'utiliser TLS |
|---|---|---|
| **[Haikei](https://haikei.app)** | Blobs, vagues, mesh-gradients, low-poly, stacked waves — en SVG. | Entrer `#4A8FA1` (teal) + `#ED843A` (orange) + `#FFF3EB` (crème). Idéal hero sections, fonds de slides. |
| **[SVG Backgrounds](https://www.svgbackgrounds.com)** | 250+ patterns SVG (topographie, pointillés, grilles...). | Choisir des motifs minimalistes (dots, subtle lines). Couleur → teal 10% opacity sur fond crème. |
| **[Paaatterns](https://products.ls.graphics/paaatterns/)** | Patterns géométriques modulaires, téléchargeables SVG/PNG. | Pour des fonds de cards ou sections "Education". |
| **[Visiwig](https://www.visiwig.com/patterns/)** | Patterns vectoriels simples. Gratuit + usage commercial. | Textures discrètes pour fond de slide ou email. |
| **[Shape Divider](https://www.shapedivider.app)** | Séparateurs de sections en vague/courbe pour le web. | Transitions entre sections hero → content. |

### Dégradés de marque

Générés avec les tokens TLS. À utiliser dans les fonds sombres (hero, covers) :

```
Teal gradient  : #1F3E45 → #4A8FA1
Warm gradient  : #4A8FA1 → #ED843A
Sun gradient   : #ED843A → #F8B044
Glass teal     : rgba(74,143,161,0.12) sur fond blanc
```

## 🤖 Assets générés par IA (sur-mesure TLS)

Quand les banques de photos/illustrations ne suffisent pas, on peut **générer des assets uniques** alignés au style TLS. Avantage : cohérence totale, aucun risque de "déjà vu" sur d'autres sites.

### Illustrations

| Outil | Force | Prompt type TLS |
|---|---|---|
| **Midjourney** | Qualité très haute, style contrôlable. | `flat illustration, mentor and learner collaboration, warm teal and orange palette, modern minimalist style, Storyset-like, white background --ar 16:9` |
| **Adobe Firefly** | Intégré Creative Cloud, modèle entraîné sur assets légaux. | `professional development illustration, diverse team learning, warm colors teal orange beige, clean vector style` |
| **DALL·E 3** (via ChatGPT) | Accès simple, bon pour des concepts rapides. | `minimalist illustration of professional coaching session, teal and warm orange color scheme, flat design, no text` |

### Photos réalistes

| Outil | Usage |
|---|---|
| **Midjourney** | `Documentary photo, two professionals having a mentoring conversation, warm natural light, modern coworking space, teal accents, --ar 16:9 --style raw` |
| **Stable Diffusion (SDXL)** | Si on a besoin de beaucoup d'images (scalable). Contrôle total sur le style. |
| **Adobe Firefly (Generative Fill)** | Pour retoucher des photos stock (changer le fond, ajuster la palette). |

### Icônes custom

Si Lucide manque une icône spécifique EdTech :
- **[IconScout](https://iconscout.com)** — packs Lucide-compatibles (stroke, 24px)
- **Figma Community** — packs d'icônes outline gratuits filtrables par style

## 🎨 Palette de sélection — Cheat Sheet visuel

Utiliser ce tableau pour valider rapidement qu'un asset est compatible TLS :

| Critère | ✅ Compatible | ❌ Incompatible |
|---|---|---|
| Température lumière | Chaude (3000–4500K), naturelle | Froide, flash, studio bleu |
| Couleur dominante | Teal, crème, bois, terracotta, blanc cassé | Rouge vif, violet, bleu électrique, noir dominant |
| Composition | Ouverte, espace de respiration | Surchargée, trop dense |
| Sujet | Humain(s) en action, collaboration | Corporate rigide, trop corporate |
| Style photo | Documentaire, candid, naturel | Posé artificiellement, publicité lisse |
| Style illustration | Flat moderne, arrondi chaleureux | Cartoon exagéré, rétro, low-effort clipart |

## 📁 Organisation — Nommage & Stockage

### Structure recommandée dans ce repo

```
brand-assets/
├── photos/
│   ├── hero/           # Images larges 16:9 pour heroes/slides
│   ├── cards/          # Vignettes carrées ou 4:3 pour cards/articles
│   └── portraits/      # Portraits proches pour testimonials
├── illustrations/
│   ├── storyset/       # Illustrations Storyset recolorisées
│   ├── undraw/         # Illustrations unDraw exportées
│   └── custom/         # Assets générés sur-mesure
├── patterns/
│   ├── backgrounds/    # SVG de fond générés (Haikei etc.)
│   └── dividers/       # Séparateurs de section
├── icons/
│   └── custom/         # Icônes custom hors Lucide
└── video/
    ├── hero/           # Boucles de fond
    └── content/        # B-roll clips
```

### Convention de nommage

```
[contexte]_[sujet]_[format].[ext]

Exemples :
hero_mentoring-session_16x9.jpg
card_team-collaboration_square.jpg
illustration_learning-journey_undraw.svg
pattern_blob-teal-orange_bg.svg
```

### Dans Canva Brand Kit

Ajouter les assets validés dans :
- **Photos** → photos prêtes à l'emploi (héros, cards, portraits)
- **Graphics** → illustrations SVG, patterns, icônes custom
- **Videos** → clips hero et b-roll approuvés

Nommer chaque asset avec le même schéma que ci-dessus pour les retrouver rapidement.

## 🚀 Quick-Start — Première session de sourcing (2h)

### Sprint #1 — Photos (45 min)
1. Aller sur **Pexels** → rechercher `mentorship warm light`, `team learning workshop`, `coaching session hands`
2. Télécharger 15–20 candidates (résolution max)
3. Filtrer selon la cheat sheet ci-dessus → garder 6–8
4. Ranger dans `brand-assets/photos/` selon le bon dossier
5. Ajouter dans Canva Brand Kit → section **Photos**

### Sprint #2 — Illustrations (30 min)
1. Aller sur **unDraw** → coller `#4A8FA1` dans le color picker
2. Rechercher : `learning`, `mentoring`, `progress`, `collaboration`, `goals`
3. Télécharger 8–10 SVG (format SVG, pas PNG)
4. Ranger dans `brand-assets/illustrations/undraw/`
5. Ajouter dans Canva Brand Kit → section **Graphics**

### Sprint #3 — Fonds & patterns (30 min)
1. Aller sur **Haikei** → type "Blobs"
2. Configurer couleurs : `#4A8FA1`, `#ED843A`, `#FFF3EB`
3. Générer 3–4 variations (opacité différente)
4. Exporter en SVG
5. Idem avec "Layered Waves" et "Mesh Gradient"
6. Ranger dans `brand-assets/patterns/backgrounds/`

### Sprint #4 — Curation finale (15 min)
- Vérifier la cohérence ensemble (tout dans un même dossier Finder)
- Supprimer les doublons ou les assets qui "jurent"
- Vérifier licences de tout
- Commit dans le repo + upload dans Canva Brand Kit
