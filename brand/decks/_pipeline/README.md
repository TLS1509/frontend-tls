# Pipeline gabarits TLS

Génération du .pptx on-brand (tokens code → slides).

```bash
npm install pptxgenjs react react-dom react-icons sharp
node build.js            # → TLS-gabarits-echantillon.pptx
python3 embed_fonts.py   # → version avec League Spartan + Nunito embarquées
```

Fonts : League Spartan + Nunito (Google Fonts OFL, instances Regular/Bold dans assets/).
