Liquid Mastery · v1.0

Trois lumières contre le silence.

Le langage visuel de The Learning Society. Teal côtier pour la progression concentrée, ambre chaud pour le contact humain, or solaire pour la pratique validée. Tout le reste recule dans la profondeur — pour que les trois lumières restent lumineuses.

Direction créative

La doctrine.

Cinq convictions qui font que ce système ne pouvait être conçu que pour cette organisation — impossible à confondre, impossible à imiter sans comprendre les décisions derrière chaque geste.

Foundations

Couleur.

Trois présences saturées et souveraines. Le reste est ink neutre ou profondeur navy. Pas de violet. Pas de vert. Pas de cyan qui ne soit pas teal côtier.

Sémantique — muted, jamais les couleurs web brutes

Foundations

Typographie.

League Spartan parle à 64 points minimum — puis le silence. Nunito porte le corps de texte. Playfair pour les surfaces éditoriales et réflexives. JetBrains Mono pour les coordonnées.

Espacement & rayons.
